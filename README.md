# About the Project

<!-- Summary -->

### Summary

Cold Email Pipeline is a localhost-only project designed to help generate personalized cold emails for client lead generation across any project.

Workflow for managing prospects, crafting personalized email templates, and tracking outreach campaigns with SQLite database management.

## Management

The application provides a web interface for:

- Adding and editing prospect information
- Creating email templates
- Tracking campaign performance (not round trip)
- Exporting data for analysis
